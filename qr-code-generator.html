Technical Details & Optimization

Our generator creates Static QR Codes. This means the information is encoded directly into the pattern of the image. This ensures that the code will work indefinitely, as there is no redirecting server in between. The codes are generated using the standard ISO/IEC 18004 specification, ensuring compatibility with all modern smartphone cameras and scanner apps.
